# 💰 Expense Tracker

A modern expense tracking web application built using Next.js that helps users manage, visualize, and analyze their income and expenses efficiently.

---

## 🚀 Features

- ➕ Add and ❌ delete transactions
- 📂 Category-based expense tracking (Food, Travel, Shopping, Salary, Others)
- 🧠 Smart input handling (custom description only for "Others")
- 📅 Date tracking for each transaction
- 📊 Real-time balance calculation (Income, Expense, Total)
- 📈 Data visualization using pie chart
- 💾 Persistent data using localStorage
- 🎨 Clean and responsive UI

---

## 🛠️ Tech Stack

- **Frontend:** Next.js, React
- **Visualization:** Recharts
- **Storage:** localStorage
- **Styling:** CSS (custom UI)

---

## 📊 How It Works

- Users enter an amount and select a category
- If "Others" is selected, a custom description is added
- Transactions are stored locally in the browser
- Dashboard updates automatically
- Chart visualizes income vs expenses
